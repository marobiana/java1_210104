## 비교 연산자의 종류
일반적으로 크다, 작다, 같다, 다르다 를 표현하는 방식으로 수학에서 표시하는 기호와 유사하다.

|비교 연산자|	의미|
|------|------|
|==|	같다|
|!=	|다르다|
|>	|크다 (초과)|
|<	|작다(미만)|
|>=	|크거나 같다 (이상)|
|<=	|같거나 같다 (이하)|

## 논리 연산자의 종류
|논리 연산자|사용 예|	의미|
|------|------|------|
|&&|a && b|**"그리고"** 논리곱. a와 b가 모두 참일 때 참|
|\|\|	|a \|\| b|**"또는"** 논리합. a나 b 중 한쪽만 참이여도 참|
|!|!x|논리부정. 참과 거짓을 뒤집음|
